# 医药信息管理系统 python620

## 项目概述

医药信息管理系统是一款针对药品、顾客及经办人信息进行管理的软件。系统支持信息的查询、录入、删除、修改以及数据报表生成和信息浏览等功能。

## 技术栈

- 前端：HTML、CSS、JavaScript、jQuery
- 后端：Python、Django
- 数据库：MySQL
- 开发工具：PyCharm、Navicat

## 运行环境

- Python 3.0以上
- MySQL 8.0

## 功能模块

- 药品信息管理
  - 信息查询
  - 信息录入
  - 信息删除
  - 信息修改
- 顾客信息管理
  - 信息查询
  - 信息录入
  - 信息删除
  - 信息修改
- 经办人信息管理
  - 信息查询
  - 信息录入
  - 信息删除
  - 信息修改
- 数据报表
- 信息浏览

## 系统角色

- 管理员：负责系统的维护和用户管理
- 普通用户：执行信息查询、录入、删除和修改操作
